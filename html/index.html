🔒 Nginx Server Header Removal

✅ Server header removal is active!

설정된 보안 헤더:

  • server_tokens off - nginx 버전 숨김
  • more_clear_headers Server - Server 헤더 완전 제거
  • X-Frame-Options - 클릭재킹 방지
  • X-Content-Type-Options - MIME 타입 스니핑 방지
  • X-XSS-Protection - XSS 공격 방지

테스트 방법:

브라우저 개발자 도구의 Network 탭에서 Response Headers를 확인하거나, 다음 명령어를 사용하세요:

curl -I http://localhost:8080
